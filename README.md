# Boot.dev Project Portfolio

This repository houses various backend development projects completed through the Boot.dev curriculum. It serves as a comprehensive collection of my programming progress and technical skills.

## Projects

### Bookbot
The Bookbot application is a Python tool designed to analyze books or other text files for statistical data.

* **Word Counting:** The software calculates the total number of words found in a file.
* **Character Frequency:** It tracks how many times each letter of the alphabet appears throughout the text.
* **Case Insensitive Logic:** All text is converted to lowercase to ensure accurate letter counts regardless of capitalization.
* **Terminal Reporting:** The tool prints a sorted list of character frequencies to the console for easy analysis.



### Asteroids
A classic arcade-style Asteroids game built in Python using the Pygame library to practice Object-Oriented Programming (OOP) and 2D vector mathematics.

* **Vector Physics:** Uses 2D vectors to handle smooth movement, rotation, and velocity for game objects.
* **OOP Architecture:** Features a clean class structure where the player, asteroids, and shots inherit from a core game object class.
* **Game Loop & Collisions:** Implements a continuous loop to update positions, handle user inputs, and check for object collisions every frame.
