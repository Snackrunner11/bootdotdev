# Asteroids

A classic arcade-style Asteroids game built in Python using the Pygame library. This project is part of the Boot.dev curriculum, designed to practice Object-Oriented Programming (OOP) concepts, game loops, and 2D vector mathematics.

## Features

* **Classic Gameplay:** Control a spaceship, shoot asteroids, and avoid collisions.
* **Vector Physics:** Smooth movement, rotation, and velocity handling using 2D vectors.
* **Object-Oriented Design:** Clean separation of game objects (Player, Asteroids, Shots) inheriting from a base `Sprite` class.

Before running the game, make sure you have Python 3 and Pygame installed.
